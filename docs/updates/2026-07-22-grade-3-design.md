# 2026-07-22 三年級單元設計補完

## 摘要

本次完成三年級自然拼讀單元的主要學習路徑設計，涵蓋地圖頁、拼讀課、閱讀流暢度頁與進度頁，並修正音段卡片在尚未解鎖時未顯示完整發音標記的問題。

## 更新內容

- 三年級地圖頁：加入單元卡片狀態、單元進度條、課程節點、下一課與複習提示。
- 三年級拼讀課：加入課程概覽與單字練習路徑，讓學生能看見目前練習位置。
- 三年級閱讀流暢度：加入閱讀概覽與逐詞閱讀路徑，強化句子閱讀流程。
- 全站進度頁：加入三年級六個單元的完成比例、完成課程數與最近練習日期。
- 發音顯示：鎖定中的音段也會顯示音標，例如 `ai /eɪ/`、`n /n/`，不再以「等一下」取代。
- 測試：新增三年級概覽、閱讀路徑與進度頁測試，並更新既有進度頁測試。
