# 2026-07-22 三年級發音完整呈現檢查

## 摘要

本次逐一檢查三年級所有拼讀課程的音段卡片，確認每個音段都會在畫面上呈現正確發音標記，包含尚未解鎖的音段與多音節小段。

## 修正內容

- 修正三年級複合字與長字闖關課的小段發音標記。
- `sun` 改為 `/sʌn/`。
- `set` 改為 `/sɛt/`。
- `cup` 改為 `/kʌp/`。
- `cake` 改為 `/keɪk/`。
- `rain` 改為 `/reɪn/`。
- `coat` 改為 `/koʊt/`。
- 新增完整測試，覆蓋三年級每一課、每一個單字、每一個音段，確認非靜音音段都以 `/.../` 發音格式呈現，靜音音段顯示「不發音」。
