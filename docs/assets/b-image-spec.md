# B 字母圖片資產規格

## 目的

本文件定義 B 字母課程第一批圖片資產的風格、尺寸、構圖、檔名、路徑、alt 文字與驗收標準。

本階段只建立規格與清單，不生成圖片、不下載圖片、不開放 B 課遊戲入口。

## 適用範圍

適用於 B 課程中的：

- 核心單字圖：ball、bat、bus。
- 句子情境圖：a ball.、The bat is big.、The bus is blue.
- 活動選項圖：聽音選圖、聲音分類、看字選圖與複習活動。
- 共用干擾圖：cat、dog、fish。

## 採用視覺風格

本專案 B 課程圖片採用：

> 柔和 3D 卡通教育插畫，圓潤造型、明亮自然色、乾淨背景、主體置中、表情友善、適合兒童學習網站。

選擇理由：

- 需延續 A 課既有圖片語言，保持 A 到 F 課程的一致性。
- B 課單字以常見生活物件為主，畫面應清楚、直接，不需要複雜故事背景。
- 圖片會重用到課程頁、遊戲與複習，因此主體辨識優先於裝飾性。

## 尺寸與比例

### 核心單字卡圖片

- 比例：1:1
- 來源尺寸：1024x1024
- 格式：WebP
- 用途：課程頁單字卡、聽音選圖、聲音分類、複習活動。

### 句子情境圖

- 比例：4:3
- 來源尺寸：1200x900
- 格式：WebP
- 用途：短句跟讀、句子配圖、後續複習。

## 構圖原則

- 主體置中，完整顯示，四周保留足夠邊界。
- 核心單字圖只呈現一個主要目標物件。
- 句子情境圖可有少量場景，但不得喧賓奪主。
- 不在圖片中顯示英文字母、英文單字、中文、數字標籤。
- bat 在本課固定指球棒，不是蝙蝠。

## 背景原則

### 核心單字圖

- 使用簡單純色、柔和漸層或極簡背景。
- 主體與背景要有足夠對比。
- 不使用球場、車站、街景等大量背景細節。

### 句子情境圖

- 可使用草地、地面、簡單街道或乾淨背景。
- 場景只服務句意，不加入無關人物與物件。

### 活動圖片

- 正確圖與干擾圖需要維持接近的視覺複雜度與完成度。
- 不可只有正確答案畫得精緻，干擾圖卻明顯較弱。

## 色彩原則

- 使用明亮自然色，避免刺眼高飽和。
- ball 可用紅、黃、藍等兒童友善配色，但球體要清楚簡單。
- bat 建議用木頭暖棕色。
- bus 固定藍色或亮藍色，方便句子圖延伸 `blue`。
- 干擾圖顏色可活潑，但不能搶過主要教學焦點。

## 角色與物件一致性

同一單字若在多張圖出現，需保持：

- 外型一致
- 主色一致
- 比例一致
- 畫風一致

固定規則：

- ball：單一彩色球，外觀圓潤乾淨。
- bat：單一球棒，清楚可辨識為球棒。
- bus：藍色公車，側面或三分之二角度，輪子明顯。

## 禁止內容

所有圖片禁止出現：

- 英文字母或單字
- 中文
- 水印
- 商標
- 品牌包裝
- 過度寫實質感
- 複雜背景
- 多個競爭焦點
- 恐怖或攻擊畫面
- 把 bat 畫成蝙蝠

## 目錄與檔名規則

建議目錄：

```text
public/
└─ assets/
   └─ images/
      ├─ courses/
      │  └─ b/
      │     ├─ b-ball-core.webp
      │     ├─ b-bat-core.webp
      │     ├─ b-bus-core.webp
      │     ├─ b-sentence-a-ball.webp
      │     ├─ b-sentence-big-bat.webp
      │     └─ b-sentence-blue-bus.webp
      └─ shared/
         ├─ shared-cat-core.webp
         ├─ shared-dog-core.webp
         └─ shared-fish-core.webp
```

檔名規則：

- 課程專屬圖片：`b-[target]-[purpose].webp`
- 句子圖：`b-sentence-[short-slug].webp`
- 共用干擾圖：`shared-[target]-core.webp`

## Alt 文字規則

- 使用繁體中文與台灣用語。
- 描述實際可見主體，不直接說明測驗答案。
- 避免過長，但要足夠區分物件。

範例：

- `一顆適合兒童辨識的彩色球`
- `一支木製球棒放在簡單背景前`
- `一台側面清楚可見的藍色公車`

## 核心單字圖片規格

### ball

- 圖片內容：一顆單一彩色球。
- 不可包含：人物、球場、品牌圖案、多顆球。
- 構圖：球要完整且容易辨識。

### bat

- 圖片內容：一支單一球棒。
- 不可包含：球員、手套、球帽、蝙蝠。
- 構圖：主體單純，球棒輪廓清楚。

### bus

- 圖片內容：一台藍色公車。
- 不可包含：擁擠乘客、複雜城市背景、品牌標誌。
- 構圖：車身與輪子清楚，適合在小卡片中辨識。

## 句子情境圖片規格

### a ball.

- 需求：單一球放在簡單地面、草地或室內地板上。
- 不加入人物。
- 保持畫面簡潔。

### The bat is big.

- 需求：一支看起來較大的球棒。
- 可透過旁邊小石頭或簡單物件做比例參照。
- 不畫成蝙蝠。

### The bus is blue.

- 需求：一台藍色公車停在簡單街道旁。
- 背景簡單，重點是 `bus` 與 `blue`。
- 不加入大量街景或乘客。

## 與 B 課程資料的對應

目前 `src/courses/data/b.ts` 已先放入規劃中的檔名、路徑、alt 與 placeholder 狀態。

本階段不建立實體圖片檔，避免課程頁誤判為正式完成。

## 圖片驗收標準

- 主體正確。
- 主體完整，不裁切。
- 無文字、水印、商標。
- 兒童友善、畫風一致。
- 可在小卡片尺寸下快速辨識。
- 干擾圖與正確圖品質一致。

## Non-scope

本文件不處理：

- 實際圖片生成
- 實際 WebP 檔案建立
- B 課正式音訊
- B 課遊戲入口開放
- C 到 F 圖片規格
- G 到 Z
