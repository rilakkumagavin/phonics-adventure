# G 課程圖片資產規格

## 任務邊界

本規格涵蓋 G 課程第一批三張核心單字圖。句子階段暫時重用核心圖片，避免在驗證硬 G 學習流程前擴大資產數量。

## 共通視覺規格

- 兒童友善的柔和 3D 故事書插畫。
- 主體置中、輪廓完整、背景簡單且光線明亮。
- 不含文字、字母、浮水印、邊框或品牌標誌。
- 圖片比例為 1:1，縮小至課程卡片時仍能快速辨識。
- 避免驚嚇表情、危險情境與不必要的小物件。

## 核心單字圖

| 檔名                  | 主體       | 關鍵要求                               | alt                                      |
| --------------------- | ---------- | -------------------------------------- | ---------------------------------------- |
| `g-goat-core.webp`    | 白色小山羊 | 全身入鏡，角與四隻蹄清楚，表情友善     | 一隻友善的白色小山羊站在簡單背景前       |
| `g-gift-core.webp`    | 禮物盒     | 單一盒子，紅色蝴蝶結與盒身輪廓清楚     | 一個綁著紅色蝴蝶結的藍綠色禮物盒         |
| `g-gorilla-core.webp` | 幼年大猩猩 | 全身入鏡，不露牙、不做威嚇姿勢         | 一隻表情友善的幼年大猩猩坐在簡單背景前   |

## 驗收檢查

1. 一眼可辨識 goat、gift、gorilla。
2. 三張圖片的插畫風格、光線與背景複雜度一致。
3. gorilla 的表情友善，不造成兒童壓力。
4. 圖中沒有文字、變形肢體或多餘主體。
5. 圖片裁切與縮放後，主體仍完整清楚。
