# F 課程圖片資產規格

## 任務邊界

本規格只涵蓋 F 課程的三張核心單字圖與三張句子情境圖。圖片需逐批生成、人工檢查後才可將課程資料狀態改為 `ready`。

## 共通視覺規格

- 兒童友善的柔和 3D 故事書插畫。
- 主體清楚、輪廓完整、背景簡單。
- 光線明亮，色彩自然，不使用單一色系淹沒主體。
- 不含文字、字母、浮水印、邊框或品牌標誌。
- 核心單字圖使用 1:1；句子圖使用 4:3。
- 避免驚嚇表情、危險情境與不必要的小物件。

## 核心單字圖

| 檔名               | 主體       | 關鍵要求                               | alt                              |
| ------------------ | ---------- | -------------------------------------- | -------------------------------- |
| `f-fish-core.webp` | 單一小魚   | 側面全身、魚鰭與尾巴清楚，不加其他動物 | 一條友善的小魚在簡單水藍背景前   |
| `f-fan-core.webp`  | 桌上型風扇 | 正面、護網與三片扇葉清楚，保持關閉     | 一台有三片扇葉的桌上型電風扇     |
| `f-frog-core.webp` | 綠色青蛙   | 全身蹲姿、眼睛與後腿清楚，表情友善     | 一隻友善的綠色青蛙蹲在簡單背景前 |

## 句子情境圖

| 檔名                           | 句子               | 關鍵要求                                     | alt                                  |
| ------------------------------ | ------------------ | -------------------------------------------- | ------------------------------------ |
| `f-sentence-a-fish.webp`       | A fish.            | 單一小魚在清澈水中游動                       | 一條小魚在清澈水中游動的句子圖片     |
| `f-sentence-fast-fan.webp`     | The fan is fast.   | 以扇葉動態模糊或氣流線呈現快速旋轉，不加文字 | 一台桌上型電風扇快速轉動的句子圖片   |
| `f-sentence-jumping-frog.webp` | The frog can jump. | 青蛙離地跳躍，動作自然、落點安全             | 一隻綠色青蛙正在草地上跳躍的句子圖片 |

## 驗收檢查

1. 一眼可辨識 fish、fan、frog。
2. 三張核心圖風格、光線與背景複雜度一致。
3. 風扇不會被誤認為螺旋槳，扇葉沒有危險裸露。
4. 圖中沒有文字、變形肢體或多餘主體。
5. 裁切後主體仍完整，縮小至課程卡片仍清楚。
