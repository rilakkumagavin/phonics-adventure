# I 課程圖片資產規格

## 任務邊界

本規格涵蓋 I 課程的 igloo、insect、ink 三張核心單字圖。句子階段暫時重用核心圖片。

## 共通視覺規格

- 兒童友善的柔和 3D 故事書插畫。
- 主體置中、完整、背景簡單，圖片比例為 1:1。
- 不含文字、字母、浮水印、邊框或品牌標誌。
- 昆蟲不呈現驚嚇細節；墨水不灑出。

## 核心單字圖

| 檔名                    | 主體     | 關鍵要求                         | alt                                  |
| ----------------------- | -------- | -------------------------------- | ------------------------------------ |
| `i-igloo-core.webp`     | 白色冰屋 | 圓頂、冰磚與入口清楚             | 一座由白色冰磚堆成的小冰屋           |
| `i-insect-core.webp`    | 紅色瓢蟲 | 單一昆蟲，觸角與腳清楚，表情友善 | 一隻表情友善的紅色瓢蟲               |
| `i-ink-core.webp`       | 藍色墨水 | 單一透明瓶，瓶身完整，沒有標籤   | 一瓶裝著鮮藍色墨水的透明玻璃瓶       |

## 驗收檢查

1. 一眼可辨識 igloo、insect、ink。
2. 沒有文字、多餘主體或危險情境。
3. 三張圖縮小至課程卡片後仍清楚。
