# H 課程圖片資產規格

## 任務邊界

本規格涵蓋 H 課程第一批三張核心單字圖。句子階段暫時重用核心圖片，先驗證 H 的聽、說、讀流程。

## 共通視覺規格

- 兒童友善的柔和 3D 故事書插畫。
- 主體置中、輪廓完整、背景簡單且光線明亮。
- 不含文字、字母、浮水印、邊框或品牌標誌。
- 圖片比例為 1:1，縮小至課程卡片時仍能快速辨識。
- 動物表情平靜友善，不呈現攻擊或危險情境。

## 核心單字圖

| 檔名                 | 主體       | 關鍵要求                           | alt                                      |
| -------------------- | ---------- | ---------------------------------- | ---------------------------------------- |
| `h-hat-core.webp`    | 紅色帽子   | 單一帽子，帽身與帽簷完整清楚       | 一頂有藍綠色縫線的紅色帽子               |
| `h-hen-core.webp`    | 金棕色母雞 | 全身入鏡，雞冠、翅膀與腳清楚       | 一隻表情友善的金棕色母雞站在簡單背景前   |
| `h-hippo-core.webp`  | 灰藍小河馬 | 全身入鏡，嘴巴閉合，不露大牙       | 一隻表情友善的灰藍色小河馬站在簡單背景前 |

## 驗收檢查

1. 一眼可辨識 hat、hen、hippo。
2. 三張圖片的插畫風格、光線與背景複雜度一致。
3. 動物肢體完整、表情友善，不造成兒童壓力。
4. 圖中沒有文字、變形肢體或多餘主體。
5. 圖片裁切與縮放後，主體仍完整清楚。
