# E 課程圖片資產規格

## 任務邊界

本規格只涵蓋 E 課程的三張核心單字圖與三張句子情境圖。圖片需逐批生成、人工檢查後才可將課程資料狀態改為 `ready`。

## 共通視覺規格

- 兒童友善的柔和 3D 故事書插畫。
- 主體清楚、輪廓完整、背景簡單。
- 光線明亮，色彩自然，不使用單一色系淹沒主體。
- 不含文字、字母、浮水印、邊框或品牌標誌。
- 核心單字圖使用 1:1；句子圖使用 4:3。
- 避免驚嚇表情、危險情境與不必要的小物件。

## 核心單字圖

| 檔名                   | 主體         | 關鍵要求                         | alt                          |
| ---------------------- | ------------ | -------------------------------- | ---------------------------- |
| `e-egg-core.webp`      | 單顆白色雞蛋 | 完整、置中，不出現破蛋或其他食物 | 一顆白色雞蛋放在簡單背景前   |
| `e-elephant-core.webp` | 友善的小象   | 全身可見，象鼻與耳朵清楚         | 一隻友善的小象站在簡單背景前 |
| `e-elbow-core.webp`    | 彎曲的手臂   | 聚焦手肘，不使用醫療標示或文字   | 一個彎曲手臂的手肘部位       |

## 句子情境圖

| 檔名                           | 句子                 | 關鍵要求                             | alt                                |
| ------------------------------ | -------------------- | ------------------------------------ | ---------------------------------- |
| `e-sentence-an-egg.webp`       | An egg.              | 單顆蛋放在簡單桌面，主體清楚         | 一顆雞蛋放在簡單桌面上的句子圖片   |
| `e-sentence-big-elephant.webp` | The elephant is big. | 以構圖呈現大象體型，但保持友善       | 一隻體型較大的友善大象在簡單背景前 |
| `e-sentence-touch-elbow.webp`  | Touch your elbow.    | 一位兒童自然碰觸自己的手肘，動作清楚 | 一位兒童輕輕碰著自己手肘的句子圖片 |

## 驗收檢查

1. 一眼可辨識 egg、elephant、elbow。
2. 三張核心圖風格、光線與背景複雜度一致。
3. 手肘圖不會被誤認為手腕、肩膀或醫療圖。
4. 圖中沒有文字、變形肢體或多餘主體。
5. 裁切後主體仍完整，縮小至課程卡片仍清楚。
